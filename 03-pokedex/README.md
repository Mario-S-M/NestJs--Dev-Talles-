# Ejecutar en desarrollo

1. Clonar el rpositorio
2. Ejecutar
```
npm i
```
3. Tener NestCLI instalado
```
npm i -g @nestjs/cli
```
3. Levantar la base de datos
```
docker-compose up -d
```
4. Stack usadno
* MongoDB
* Nest
